ツイステ魔法石計算機 開発記事

概要

「ツイステ魔法石計算機」は、Twisted Wonderland のガチャを引きたい日までにどれだけの魔法石を貯められるかをシミュレーションする Web アプリです。

現在の所持石・鍵、誕生日ボーナス、月次配布鍵、30daysパス、初回限定＆有償パッケージなどを総合的に計算し、目標の連数に届かない場合は最適な購入プランを提案します。

開発環境とフレームワーク

  • JavaScript + React（Function Components + Hooks）
  • Vite（高速開発サーバ＆ビルド）
  • CSS モジュール
  • GitHub Pages でホスティング

こだわったポイント

  • 状態管理はすべて useState でシンプルに。
  • 計算ロジックは副作用なしの純粋関数にまとめ、テストや再利用を容易に。
  • 入力→結果表示の UI をカテゴリごとにコンポーネント化し、直感的な操作感を実現。
  • 購入プランの最適化アルゴリズムは再帰探索＋枝刈りで高パフォーマンスを維持。

月次配布鍵・ショップ鍵・イベント鍵

今月分：配布鍵3個、ショップ（10連鍵+鍵3個）実質90魔石、イベント鍵3個（オプション）
来月以降：月ごとの合計魔石価値を月数×（配布90+ショップ90+（イベント90））で算出

30daysパス累計魔石

function calculate30DaysPassActualStones(count, daysUntilTarget) {
  let total = 0;
  for (let i = 0; i < count; i++) {
    const offset = i * 30;
    if (offset < daysUntilTarget) {
      total += 48; // 購入即時
      total += Math.min(daysUntilTarget - offset, 30) * 5;
    }
  }
  return total;
}

購入提案アルゴリズム

初回限定パック、30daysパス、1日限定パック、通常パックを効率（魔石/円）順にソートし、
再帰的に「何個買うか」を全探索＋枝刈りし、必要石数を満たす最安組み合わせを出力。
